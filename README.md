## ✨ Features

-   **Interactive Parallax Effect:** Background elements subtly respond to mouse movement, adding depth and engagement.
-   **Dynamic Text Animation:** The "Page not found" message features a typewriter-like animation for a smooth, captivating display.
-   **Responsive Design:** Optimized for various screen sizes, ensuring a consistent experience on desktops, tablets, and mobile devices.
-   **"Go Home" Button:** A clear call-to-action to help users navigate back to the main site.
-   **Modern Aesthetic:** Clean and visually appealing design using plain HTML, CSS, and JavaScript.

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/is4r0/404.git
    cd 404
    ```

2.  **Open in your browser**
    Simply open the `index.html` file in your preferred web browser:
    ```bash
    # For macOS/Linux (might vary based on your default browser)
    open index.html

    # For Windows
    start index.html
    ```
    Alternatively, you can drag and drop `index.html` into your browser window.
